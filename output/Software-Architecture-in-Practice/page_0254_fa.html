که Reduce Instance 2 نامیده می‌شود. Reduce Instance 2 داده‌ها را از تمام Partition 2s تولید شده توسط نمونه‌های map مختلف دریافت می‌کند. ممکن است چندین تکرار از reduce برای فایل‌های بزرگ وجود داشته باشد، اما این در شکل 13.14 نشان داده نشده است.

یک مسئله آموزشی کلاسیک برای map-reduce، شمارش وقوع کلمات در یک سند است. این مثال را می‌توان با یک تابع map واحد انجام داد. سند، مجموعه داده است. تابع map، هر کلمه را در سند پیدا می‌کند و یک جفت <word, 1> را برای هر کدام خروجی می‌دهد. به عنوان مثال، اگر سند با کلمات "Having a whole book . . . ," شروع شود، نتایج اول map به این صورت خواهد بود:




در عمل، "a" یکی از کلماتی خواهد بود که توسط map فیلتر شده است.

Pseudocode برای map ممکن است به این صورت باشد:

map(String key, String value):
// کلید: نام سند
// مقدار: محتویات سند
for each word w in value:
