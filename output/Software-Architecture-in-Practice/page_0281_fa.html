14.2  Quality Attribute Checklists

برای برخی از quality attributes، checklists وجود دارد تا architect را قادر به test کردن compliance یا هدایت architect هنگام اتخاذ تصمیمات طراحی کند. Quality attribute checklists می‌توانند از کنسرسیوم‌های صنعتی، از سازمان‌های دولتی، یا از سازمان‌های خصوصی به دست آیند. در سازمان‌های بزرگ ممکن است در داخل خانه توسعه یابند.

این checklists می‌تواند به یک یا چند quality attributes خاص باشد. checklists برای safety، security، و usability رایج هستند. یا آنها ممکن است بر یک domain خاص متمرکز شوند. checklists security برای صنعت مالی، کنترل فرآیند صنعتی، و بخش انرژی الکتریکی وجود دارد. آنها حتی ممکن است بر روی برخی از جنبه‌های خاص یک quality attribute واحد متمرکز شوند: cancel برای usability، به عنوان مثال.

برای اهداف صدور گواهینامه یا مقررات، checklists می‌توانند توسط حسابرسان و همچنین توسط architect استفاده شوند. به عنوان مثال، دو مورد از موارد موجود در checklist Payment Card Industry (PCI) این است که فقط شماره کارت‌های اعتباری را در یک فرم رمزگذاری شده حفظ کنید و هرگز کد security را از پشت کارت اعتباری حفظ نکنید. یک حسابرس می‌تواند درخواست کند که داده‌های کارت اعتباری ذخیره‌شده را بررسی کند تا مشخص کند که آیا رمزگذاری شده است یا خیر. حسابرس همچنین می‌تواند طرحواره داده‌های ذخیره‌شده را بررسی کند تا ببیند آیا کد security گنجانده شده است یا خیر.

این مثال نشان می‌دهد که طراحی و تجزیه و تحلیل اغلب دو روی یک سکه هستند. با در نظر گرفتن انواع تجزیه و تحلیل‌هایی که یک سیستم (در این مورد، یک حسابرسی) مورد بررسی قرار می‌گیرد، architect به سمت اتخاذ تصمیمات معماری اولیه مهم هدایت می‌شود (تصمیماتی که حسابرسان می‌خواهند پیدا کنند).

Security checklists معمولاً دارای process components سنگینی هستند. به عنوان مثال، یک security checklist ممکن است بگوید که باید یک سیاست امنیتی صریح در داخل یک سازمان و یک cognizant security officer برای اطمینان از compliance با این سیاست وجود داشته باشد. آنها همچنین دارای technical components هستند که architect باید برای تعیین پیامدهای معماری سیستمی که طراحی یا ارزیابی می‌شود، بررسی کند. به عنوان مثال، مورد زیر از یک security checklist است که توسط گروهی که توسط یک سازمان تولیدکنندگان و توزیع‌کنندگان برق منشور شده است، ایجاد شده است. این مربوط به سیستم‌های تعبیه‌شده‌ای است که برق را به خانه شما می‌رسانند:
