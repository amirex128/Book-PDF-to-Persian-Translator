Security

Checklists برای security توسط گروه‌های مختلف برای domains مختلف تولید شده‌اند. به عنوان مثال ببینید:

  • کارت‌های اعتباری، تولید شده توسط Payment Card Industry: www.pcisecuritystandards.org/security_standards/
  • Information security، تولید شده توسط National Institute of Standards and Technology (NIST): [NIST 09].
  • Electric grid، تولید شده توسط Advanced Security Acceleration Project for the Smart Grid: www.smartgridipedia.org/index.php/ASAP-SG
  • Common Criteria. یک استاندارد بین‌المللی (ISO/IEC 15408) برای گواهینامه computer security: www.commoncriteriaportal.org

Testability

کار در اندازه‌گیری testability از دیدگاه معماری شامل اندازه‌گیری testability به‌عنوان پیچیدگی اندازه‌گیری‌شده یک نمودار وابستگی class است که از نمودارهای UML class به‌دست‌آمده است، و شناسایی نمودارهای class است که می‌تواند منجر به کدی شود که test کردن آن دشوار است [Baudry 05]؛ و اندازه‌گیری controllability و observability به‌عنوان تابعی از data flow [Le Traon 97].
