7.4.1 The OrderHistoryEventHandlers module

این ماژول شامل event handlers است که events را مصرف می‌کنند و جدول DynamoDB را به‌روزرسانی می‌کنند. همانطور که در لیست زیر نشان داده شده است، event handlers متدهای ساده‌ای هستند. هر متد یک خطی است که یک متد OrderHistoryDao را با آرگومان‌هایی که از event مشتق شده‌اند، فراخوانی می‌کند.
