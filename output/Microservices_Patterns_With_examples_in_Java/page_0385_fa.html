Developing secure services

The sequence of events for API clients به شرح زیر است:

  1. A client یک request حاوی credentials را انجام می‌دهد.
  2. The API gateway credentials را authenticate می‌کند، یک security token ایجاد می‌کند و آن را به service یا services منتقل می‌کند.

The sequence of events for login-based clients به شرح زیر است:

  1. A client یک login request حاوی credentials را انجام می‌دهد.
  2. The API gateway یک security token را برمی‌گرداند.
  3. The client security token را در requests که operations را فراخوانی می‌کنند، قرار می‌دهد.
  4. The API gateway security token را تأیید می‌کند و آن را به service یا services forward می‌کند.
