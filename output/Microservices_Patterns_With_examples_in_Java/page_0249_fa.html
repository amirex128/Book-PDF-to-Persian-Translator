Summary

  • Event sourcing یک راه ساده برای پیاده‌سازی sagas مبتنی بر choreography است. Services دارای event handlers هستند که به events منتشر شده توسط event sourcing-based aggregates گوش می‌دهند.
  • Event sourcing یک راه خوب برای پیاده‌سازی saga orchestrators است. در نتیجه، شما می‌توانید برنامه‌هایی بنویسید که منحصراً از یک event store استفاده می‌کنند.
