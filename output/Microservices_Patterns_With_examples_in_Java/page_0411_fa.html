When using a service mesh، microservice chassis بسیار ساده‌تر است. It only needs تا concerns را implement کند که به شدت با application code ادغام شده‌اند، مانند externalized configuration و health checks. The microservice chassis باید از distributed tracing با propagating distributed tracing information پشتیبانی کند، مانند B3 standard headers که در بخش 11.3.3 در مورد آن بحث کردم.
