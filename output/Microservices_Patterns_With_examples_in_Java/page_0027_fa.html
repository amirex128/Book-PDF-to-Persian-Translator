منبع عالی دیگر برای یادگیری معماری Microservice، وب‌سایت من http://microservices.io است.

این وب‌سایت نه تنها شامل زبان کامل pattern است، بلکه پیوندهایی به منابع دیگر مانند مقالات، presentations و کد نمونه نیز دارد.

درباره نویسنده

Chris Richardson یک developer و architect است. او یک Java Champion، یک JavaOne rock star، و نویسنده POJOs in Action (Manning, 2006) است، که نحوه ساخت enterprise Java applications را با فریم‌ورک‌هایی مانند Spring و Hibernate شرح می‌دهد.

Chris همچنین بنیانگذار CloudFoundry.com اصلی بود، یک Java PaaS اولیه برای Amazon EC2.

امروزه، او یک رهبر فکری شناخته شده در Microservices است و به‌طور منظم در کنفرانس‌های بین‌المللی سخنرانی می‌کند. Chris خالق Microservices.io، یک زبان pattern برای microservices است. او مشاوره و آموزش microservices را به سازمان‌های سراسر جهان که معماری microservice را اتخاذ می‌کنند، ارائه می‌دهد. Chris در حال کار بر روی سومین startup خود است: Eventuate.io، یک پلتفرم application برای توسعه microservices تراکنشی.
