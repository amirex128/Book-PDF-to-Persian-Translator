برای پاسخ به این سؤال، ابتدا تعریف می‌کنم که منظور از عبارت معماری software چیست. پس از آن، من در مورد چگونگی چند بعدی بودن معماری یک application و اینکه چگونه می‌توان آن را با استفاده از مجموعه‌ای از views یا طرح‌ها بهتر توصیف کرد، بحث می‌کنم. سپس شرح می‌دهم که چرا معماری software مهم است زیرا بر ویژگی‌های کیفیت software application تأثیر می‌گذارد.
