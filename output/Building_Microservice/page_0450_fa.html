تکثیر افقی (ادامه)

افزایش توان عملیاتی سیستم، می‌توانیم تعداد نمونه‌های Song Transcoder را افزایش دهیم.

شکل 13-2. Transcoding برای پخش جریانی که با استفاده از الگوی مصرف‌کننده رقیب مقیاس‌بندی شده است

در مورد FoodCo، از یک شکل از تکثیر افقی برای کاهش بار خواندن در پایگاه داده اصلی از طریق استفاده از read replicas استفاده شده است، همانطور که در شکل 13-3 می‌بینیم. این باعث کاهش بار خواندن در node پایگاه داده اصلی شده است، که منابع را برای رسیدگی به نوشتن آزاد می‌کند، و بسیار مؤثر عمل کرده است، زیرا بار زیادی بر روی سیستم اصلی، سنگین خواندن بود. این خواندن‌ها می‌توانستند به راحتی به این read replicas هدایت شوند، و استفاده از یک load balancer بر روی چندین read replicas رایج است.

مسیردهی به پایگاه داده اصلی یا یک read replica به صورت داخلی در microservice انجام می‌شود. برای مصرف‌کنندگان این microservice شفاف است که آیا درخواستی که ارسال می‌کنند به پایگاه داده اصلی یا read replica می‌رسد یا خیر.
